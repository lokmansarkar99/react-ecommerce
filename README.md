# ShopEasy - React E-commerce Application

A modern, responsive e-commerce web application built with React, featuring product browsing, cart management, and seamless navigation.


## Overview

ShopEasy is a fully functional e-commerce frontend application that demonstrates modern React development practices. Built without a backend, it uses local state management to provide a complete shopping experience including product browsing, search, filtering, and cart management.

### Key Highlights
-  **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
-  **Cart Management** - Add, remove, and update quantities with Context API
-  **Search & Filter** - Find products by name or sort by price
-  **Modern UI** - Clean, professional design with Tailwind CSS
-  **Fast Performance** - Optimized React components with hooks
